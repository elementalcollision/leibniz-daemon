# Independent verification — the skew-Hadamard difference family behind the order-1252 matrix (Karoui 2026)

**Date:** 2026-07-05 · **Track:** T5 (external audit) / T9 (external corpus) · **Tier:** audit ·

## Target

Karoui, A. (2026). *An explicit skew-Hadamard matrix of order 1252 via cyclotomic unions*
([arXiv:2602.16089](https://arxiv.org/abs/2602.16089); submitted to the *Journal of Combinatorial Designs*).
The paper constructs a **skew-Hadamard matrix of order 1252 = 2(5⁴ + 1)** — an order reported as **missing**
in widely-used open-source Hadamard tables — by a bordered Goethals–Seidel array over a **bordered
skew-Hadamard difference family (SHDF)** `{D₀, D₁}` in the additive group of `GF(5⁴)`, with blocks unions of
cyclotomic classes of order `N = 16`.

## The construction (as verified)

With a primitive `g ∈ GF(5⁴)×` and the order-16 cyclotomic classes `Cᵢ = gⁱ·⟨g¹⁶⟩` (each `|Cᵢ| = 39`):

> `D₀ = ⋃_{i∈I₀} Cᵢ`, `D₁ = ⋃_{i∈I₁} Cᵢ`, with `I₀ = {4,5,6,7,8,9,10,11}`, `I₁ = {0,1,2,3,4,5,6,7}`.

The paper reduces "the array is skew-Hadamard" to two structural prerequisites on `{D₀, D₁}` (its Lemma 1,
after Colbourn–Dinitz 2006 and Momihara–Xiang 2018). We check **both, exactly**, over a `GF(5⁴)` we build
from scratch (an irreducible primitive quartic over `GF(5)`, so `x` is a primitive element):

- **(S) `D₀` is skew** — for every `x ≠ 0`, exactly one of `x`, `−x` lies in `D₀` (so `|D₀| = (625−1)/2 = 312`).
  This is forced because `−1 = g³¹² ∈ C₈` (since `312 ≡ 8 mod 16`) and `I₀` is a transversal of the classes
  under `i ↦ i+8 (mod 16)`. ✓
- **(A) constant autocorrelation sum** — `A_{D₀}(w) + A_{D₁}(w) = −2` for **all 624 nonzero `w`**, where
  `A_D(w) = Σ_x ψ_D(x)ψ_D(x+w)` and `ψ_D(x) = ±1` is the difference-set indicator. ✓ (checked over every
  nonzero `w`).

**Given (S) + (A), a skew-Hadamard matrix of order `2(v+1) = 1252` exists** by the cited Goethals–Seidel
lemma — the paper's headline claim. Our independent GF(5⁴) build realizes the paper's exact index sets
`I₀ = {4..11}`, `I₁ = {0..7}` at cyclic offset `0` (the index sets are stated relative to the paper's choice
of primitive element; a different primitive `g` would relabel the 16 classes by a fixed cyclic offset — the
instrument accepts any offset of the same 8-window structure and reports which one realizes the SHDF).

## Honest scope

We verify the **difference-family prerequisites** the paper proves — the mathematically load-bearing core,
from which the order-1252 skew-Hadamard matrix follows by the standard Goethals–Seidel lemma. We did **not**
rebuild the explicit 1252×1252 matrix or re-check the paper's finite-field rank invariants (rank 1251 over
GF(2) of the tournament matrix; full rank over GF(3)/GF(5)); those need the paper's exact bordered-array
placement / artifact bundle, which is not public. The `{D₀,D₁}` verification is exact (finite-field
arithmetic, no floating point) and independent (a fresh GF(5⁴), fresh primitive element). No trust surface
touched — `tests/test_invariants.py` byte-identical; this is a pure exact-arithmetic instrument.
